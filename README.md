## Requirements

iOS 8.0+ is required.

## Installation

BotBrainFeed is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "BotBrainFeed", '~> 0.3.0'
```
